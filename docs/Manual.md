# タダサポ管理システム 操作マニュアル v1.11.3

**作成日:** 2026/01/27
**更新日:** 2026/05/10
**対象:** スタッフ / システム管理者

---

## 1. 前提条件

- Google アカウント（tadakayo.jp ドメイン）を保有していること
- 「タダメンマスタ」シートのC列に自分のメールアドレスが登録されていること
- 管理者操作には「タダメンマスタ」D列のロールが `admin` に設定されていること

---

## 2. 画面構成と表示モード

### 2-1. 表示モードの切替

画面上部に **通常 / 閲覧 / 管理** の3つのモードボタンがあります。

| モード | 表示範囲 | 操作 | 表示条件 |
|:---|:---|:---|:---|
| **通常** | 自分が担当する案件（未対応は全員共有） | 全操作可能 | 全スタッフ |
| **閲覧** | 全案件 | 読み取りのみ | 全スタッフ |
| **管理** | 全案件 | 全操作 + 管理者専用操作 | 管理者のみ |

### 2-2. 新着バッジ

各ステータスタブの右側に **`+N`** の赤いバッジが表示されます。「最後にそのタブをクリックした時点から増えた案件数」を示します。

- タブをクリックすると既読化されバッジが消えます
- 管理モードでは、ヘッダーの **Bell アイコン** でバッジのON/OFFを切り替えられます
- 閲覧モードではバッジは表示されません

## 3. 案件対応の基本フロー

### 3-1. 未対応案件を担当する

1. **「未対応」タブ** で担当したい案件を選択します
2. 年間利用回数が上限内の場合:
   - **「メール送信して担当」** — メール作成モーダルが表示されます。内容を確認・編集して送信
   - **「担当する（メールなし）」** — メール送信なしでアサインされます
3. 年間利用回数が上限に達している場合:
   - **「回数超過」** — 回数超過メールを送信し、案件を「対応不可」に変更します

### 3-2. 日程を確定する

1. 担当中の案件を選択
2. **「日程・方法を入力」** ボタンをクリック
3. 実施日時・対応方法（Google Meet / Zoom / 対面等）を入力して保存
4. Google カレンダーに予定が自動作成され、Google Meet URL が発行されます（Zoom 選択時は Zoom ミーティングが作成されます）
5. 日程確定後、確定メールの作成モーダルが自動表示されます

### 3-3. 完了報告を行う

1. 担当中の案件を選択
2. **「実施記録・完了報告」** ボタンをクリック
3. 実施内容・備考・対応ツール・サービス種別・都道府県を入力
4. ファイルを添付する場合は「ファイルを追加」または画像をドラッグ&ドロップ（最大5件）
5. **「完了にする」** または **「保存する（対応中に戻す）」** を選択して保存

### 3-4. 案件を再開する（継続支援）

完了案件に対して追加サポートを行う場合:

1. **「完了」タブ** で対象案件を選択
2. **「再開する」** ボタンをクリック
3. 対応回数が増え（2回目、3回目）、「対応中」に戻ります

再開の条件:
- 対応回数が上限（デフォルト3回）未満であること
- 今年度の利用回数が年間上限（デフォルト10回）未満であること

### 3-5. 案件をキャンセルする

1. 担当中の案件を選択
2. **「キャンセル」** ボタンをクリック
3. キャンセル理由を入力（1回目のみ）
4. 確認後キャンセルされます（担当者・サブ担当・管理者が実行可能）

## 4. メール機能

### 4-1. メールの送信モード

| モード | 用途 |
|:---|:---|
| **初回メール** | 担当アサイン時の最初のメール（テンプレートあり） |
| **新規メール** | 新しいスレッドを立てて送信 |
| **返信** | 既存のスレッドに返信 |
| **日程確定メール** | 日程確定時の通知メール（テンプレートあり） |
| **回数超過メール** | 年間上限超過時の断りメール（テンプレートあり） |

### 4-2. CC / BCC の設定

- メール作成モーダルの「CC/BCCを追加」から任意のアドレスを追加できます
- 管理者が設定した **自動CC（MAIL_FORCE_CC）** は全メールに自動付与されます（メール画面で確認可能）
- サブ担当が設定されている場合、メイン担当⇔サブ担当は自動的に CC に追加されます

### 4-3. メール下書きの保存と復元（v1.11.0）

- メール作成モーダルで **「下書き保存」** ボタンをクリックすると下書きが保存されます
- 次回同じモーダルを開いた時に「下書きを復元しますか？」と確認が表示されます
- 送信が完了すると下書きは自動的に削除されます
- 保存した下書きがある案件には **「下書きあり」** バッジが表示されます

### 4-4. メールの予約送信（v1.11.0）

- メール作成モーダルで **「予約送信」** をクリックすると日時ピッカーが表示されます
- 現在時刻の1分以上先の日時を指定して予約を確定します
- 予約したメールは指定日時に自動送信されます（最大5分の誤差あり）
- 予約がある案件には **「予約あり」** バッジが表示されます
- 案件詳細の「予約送信一覧」から予約を確認・取消できます（送信前のみ）

## 5. 検索・フィルタ

### 5-1. キーワード検索

サイドバー上部の検索欄で**常時**検索できます（事業所名・担当者名・メール・内容・種別・都道府県の横断検索）。

### 5-2. 詳細フィルタ（フィルタアイコンをクリックで展開）

| フィルタ | 内容 |
|:---|:---|
| **期間プリセット** | 今月 / 先月 / 今年度 / 前年度 をワンクリックで設定 |
| **カスタム期間** | 開始日〜終了日を直接入力 |
| **都道府県** | 絞り込み |
| **サービス種別** | 絞り込み |
| **対応ツール** | 絞り込み |
| **並び順** | 最新順 / 古い順 |
| **担当者フィルタ** | 全件 / 未割当 / 個別担当者（管理モードのみ） |

適用中のフィルタは**チップ（pill）**として表示され、個別に削除できます。

## 6. サブ担当（OJT）機能

- 案件詳細の「サブ担当を追加」から1名を指定できます（管理者・担当者が操作可）
- サブ担当はメール送信・記録修正・キャンセル等の操作が可能です
- サブ担当とメイン担当は互いにメール送信時に CC に自動追加されます

---

## 7. 管理モード操作（管理者のみ）

### 7-1. インライン編集

管理モードでは、案件詳細画面の以下の要素を**クリックするだけ**で直接編集できます。

| クリック箇所 | 操作内容 |
|:---|:---|
| ステータスバッジ（例：「対応中」） | ステータスをドロップダウンから選択して変更 |
| 担当者バッジ（例：「田中花子」） | 担当者を検索絞り込み付きドロップダウンから変更 |
| 案件回数（例：「2 / 3」） | 案件ごとの対応上限を特例変更（空欄で全体設定に戻す） |
| 今年度利用数（例：「5 / 10」） | 年間上限を特例変更（空欄で全体設定に戻す） |

ポップオーバーは **Escape キー** または外側クリックで閉じられます。

### 7-2. 権限管理ダイアログ

ヘッダーの「権限管理」ボタンから開きます。

- スタッフの名前・メールで検索できます（100名以上対応）
- ロールを「管理者 / タダメン / 無効」に変更できます
- 自分自身のロール変更は禁止されています
- 新規メンバーの追加はスプレッドシートのタダメンマスタで行います（アプリからは不可）

### 7-3. 設定管理ダイアログ

ヘッダーの「設定管理」ボタンから開きます。カテゴリ別に設定を管理できます。

| カテゴリ | 主な設定 |
|:---|:---|
| **上限設定** | 年間利用上限・案件回数上限 |
| **メール設定** | 各種メールテンプレート・自動CC |
| **連携設定** | 共有カレンダーID・添付フォルダ・Zoom API |
| **その他** | 対応ツール一覧・ツール月間上限 |

### 7-4. データ抽出

管理モード → 「データ抽出」ボタンから、案件データを CSV 形式でエクスポートできます（都道府県・事業所名・項目選択UI）。

## 8. トラブルシューティング

| 症状 | 確認事項 |
|:---|:---|
| 「アクセス権限がありません」が表示される | アクセスしているGoogleアカウントのメールアドレスが「タダメンマスタ」C列に登録されているか確認 |
| 自分の案件が見つからない | 表示モードが「通常」になっているか確認。管理者に担当者設定を依頼 |
| 編集ボタンが押せない・グレーアウトしている | 「閲覧」モードになっていないか確認。「通常」モードに切り替える |
| 新着バッジが消えない | 該当タブをクリックすると既読化される。管理モードの Bell ボタンでバッジをOFFにすることも可能 |
| 「管理」ボタンが表示されない | タダメンマスタのD列（ロール）に `admin` が設定されているか確認 |
| メールが送信されない | MAIL_DRY_RUN が `true` になっていないか設定シートを確認 |
| 予約メールが送信されない | GASエディタで `getScheduledEmailTriggerStatus()` を実行してトリガが有効か確認 |
| カレンダーに予定が作成されない | Calendar Advanced Service が有効化されているか確認。SHARED_CALENDAR_ID の設定を確認 |
